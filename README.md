# MERN Employee CRUD

A full-stack MERN app to manage employee records.  
Users can **create**, **read**, **update**, and **delete** employees.

---

## Features

- Add new employees with **name**, **position**, and **level**
- Edit existing employee records
- Delete employees
- Responsive frontend with React
- Backend API powered by Express and MongoDB
